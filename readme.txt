== Description ==

News Pro is a modern, fast, and fully responsive WordPress news theme built for professional media outlets, blogs, and digital news platforms. It features a powerful frontpage builder, posts carousel, banner sections, video showcase, editor's choice, featured posts, and much more — all configurable from the WordPress Customizer without touching a single line of code.

Key Features:
* Fully responsive and mobile-optimized
* Live Customizer with real-time preview
* Posts Carousel with Slick slider
* Banner section with Daily News, Main News, and Top Stories layouts
* Editor Choice, Featured Posts, Video Section
* Tabbed news widgets and categories section
* News highlights ticker
* Translation-ready
* Cross-browser compatible

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
